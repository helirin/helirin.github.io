Mobile app plan

Gardening application plan

This project has been carried out during UX Design Fundamentals course. I designed the functions, layout and everything else, but the flower pictures are from the internet. I used Figma to make wireframes, mockups and prototype.

According to the plan, the user can search the app for flowers with certain criteria. The user can also save a plant to his/her own shop list. Users can give recommendations about plants. Through this application user can access plant identification and plant shops.
